程序员感悟之一

更新于: 2018-02-25 | 分类于 感悟 | | 阅读次数:
字数统计: 1,078 | 阅读时长 ≈ 4

1，很多问题不见得会出在你身上，但你亦需要想法解决问题，否则就会变成你的问题

2，真正的牛人也许一辈子只投 4 次简历，这些家伙一毕业就被好公司抢走了，并且他们的雇主会给他们不赖的待遇，所以他们也不想挪窝。而「人才」市场上能找到的大多都不是什么人才，招到这帮人轻则费钱重则把你公司搞挂。

3，现在开发者们都喜欢说自己有一些伟大的想法，甚至有人发给我邮件说，让我们一起改变世界吧。但我认为，脑子里的想法在付诸实际得以成现实之前，都是一文不值的，太多人都有自己的想法，关键在于如何去实现它。就算你有很好的商业模式，也需要依靠产品支撑。而现在很多有想法的人都把自己定位为一个类似产品经理的角色，往往缺少美工，开发，测试，推广，运营的经验，但我认为，只要你想办法，这些问题其实都很好解决

4，最近一帮做技术的在微博上吵谁比谁 NB。其实做技术不像练武，非得打遍天下。能很好解决自己遇到的问题搞定公司的需求就很好；进一步，精益求精下，提升下性能重构下接口，做到优雅可读；再进一步，整理成可重用的方式发布出来，让更多人受益。真正NB是多少人因为你而活得更好，而非其他。

5，越是好的公司，越是好的项目组在招开发人员时他们面的东西越是底层。

6，编写 Java 开发包都是一群神人，拿 Java 源码来分析问题，我们无疑站在了巨人的肩膀上。所谓站得高，尿的远也。当然，所谓偷拍都是避免不了的；

7，前期追深度，否则会华而不实，后期追广度，否则会坐井观天

8，有很多的不快乐，其实是源自不满足，而不满足，很多时候是源自于心不定，而心不定则是因为不清楚究竟自己要什么，不清楚要什么的结果就是什么都想要，结果什么都没得到。

9，多认识一些人，多和比自己强的人打交道，同样能找到好的老师，不要和一群同样郁闷的人一起控诉社会，控诉老板，这帮不上你，只会让你更消极。和那些比你强的人打交道，看他们是怎么想的，怎么做的，学习他们，然后跟更强的人打交道。

10，羡慕别人有个好爸爸，没什么不可以。问题是，你的下一代，会有一个好爸爸吗？至于问到为什么不能有同样的赢面概率？我只能问：为什么物种竞争中，人和猴子不能有同样的赢面概率？物竞天择。猴子的灵魂不一定比你卑微，但你身后有几十万年的类人猿进化积淀。

11，作为职业规划师，我想告诉你关于工资的一个秘密，年薪月薪是相当有欺骗性的东西。真正起作用的，是时薪——你一定要看看自己的单位时间是否更加值钱。

12，技术之路最公平也最残酷的原因是：没有捷径，需要日积月累的积累，以及对技术持久的热情。

13，写作很重要。通过写作清晰地表达观点，这很有帮助。从某种程度上来说，写代码和写作非常相似，两种都需要你 把你的想法，通过某个结构化的方式清晰并且无歧义的表达出来。Email 当然是写不完的了，不过还有你所开发的产品特性所配套的文档、需要清晰描述 Bug 的漏洞报告、还有对你修复的 Bug 做出的解释。在大学里写作并不是重点，但是如果你写作好的话，这绝对是工作中的一个筹码。

2017时间的朋友演讲小结

更新于: 2018-02-25 | 分类于 感悟 | | 阅读次数:
字数统计: 1,430 | 阅读时长 ≈ 5

时间：2017年12月31日晚8点半，地点：上海梅赛德斯奔驰中心，寒风吹着繁华的街，场馆里坐满了对未来充满期许的观众，罗胖儿开始了他的表演
从“岁月不饶人，我们又何曾饶过岁月”开始，时间的朋友，和时间交朋友，经得起时间考验的才是朋友。

先说问题，你焦虑吗？马云都焦虑，谁不焦虑呢？特别是城市白领和高学历人群，白天在城市办公楼勤勤恳恳工作，晚上还要加班工作和学习，一刻不敢放松，大家都焦虑着，这是这个时代的通病。中国有希望吗？这是一个大命题，也是一个切身的问题，关乎每一个人。中国会遇到发展的天花板吗？随着劳动力成本不断升高，中国的人口规模还能带来有事吗？在未来的世界秩序中，中国应该处于什么样的位置呢？问题多多，答案却看起来寥寥。

说到2017年的中国公司，有两个一定躲不过，那就是腾讯和阿里，都在2017年实现了股价大增长，结合美国top5科技公司的股市表现，似乎出现了一种现象——“大者恒大，强者恒强”。同时说到2017的游戏，大家一定听说过吃鸡，吃鸡游戏的发展之快超乎所有人的想象，以至于想腾讯都快跟不上节奏；从小米枪战到荒野行动再到腾讯的吃鸡游戏，大家拼了命的开发抢上线时间，就为争取先发优势。因为，先发优势已经是一种压倒性的优势。2017年还有一个变化就是用户变化快，学习能力强，只有你想不到的，没有他们学不会的，听音频可以做到3倍速甚至更高的，简直无法想象。更恐怖的是，一旦学会了听高速，就再也不会去听低速的。正如 豌豆公主效应：体验是一种可训练的能力，一旦达成，就再也退不回去了。简单的说就是 当你觉得一个东西丑之后，还还有机会觉得它不丑吗 。用户的心智在飞速迭代。《爱丽丝漫游记》中红桃皇后说：在我们这个地方，你只有不断奔跑，才能留在原地。没有什么竞争，只有追赶用户。

动车组脑洞 很多新的消费品牌从二三线城市崛起，正如美国在二战之后输出的肯德基，麦当劳等品牌。同时带领新品牌崛起的，很多都是以前在行业里已经干得很好的老人。创业不是新兴阶层的专有权力。快递公司其实是被低估了企业，他们掌握了很多具体生活中的经验和知识。日常生意中积累的经验和知识变得价值连城。更重要的是那种你必须做一遍才能获得的知识。

热带雨林脑洞 每一波人的涌入，都需要全新的连接器。在中国，重要的不仅是技术，还有连接。新物种，往往是高价值的连接器。超级用户思维：不仅关心有多少用户，更关心有多少超级用户。“流量思维”时代，渴望吸引新用户。“超级用户”时代，服务好已有用户。

比特化脑洞 新零售——用一切手段全方位无死角地提高效率。价格战的目的，是在比较中胜出；认知战的目的，是不存在比较。互联网革命，是一场裹挟一切的变革。你不用着急出发，因为你会被抵达。效率越来越高，分工越来越细。

拔河脑洞 思考今天的中国，不能局限在中国本身。地缘政治把世界分割成地图上的样子，而基础设施把世界连接成另一个样子。世界不再只是块与块的拼图，还是基础设施连成的点线网。只拥有，不连接，就是一个资源孤岛。拔河游戏，拧一根让所有人都能参与的绳子。拳击比赛取胜的关键是让自己更强大，拔河游戏关键是让更多人站在自己这一方。

终点站脑洞 一个新维度：世界产业演化速度。命，中国的独有禀赋：超大规模性。能够帮助高科技产品实现大规模量产，这本身就是非常高的科技。

枢纽脑洞 世界格局演化对中国形成了需求。中国的枢纽地位还体现在参与塑造全球秩序。建立秩序，必须把简单的援助转化为具体的经过经贸过程。

人生算法，不断提高达成目标的概率。哪有什么直接登顶的人生？只有根据反馈不断迭代的过程。哪有什么一成不变的规律？只有可以不断增加的概率。
当成功概率很高的时刻，下最大的赌注，其余时间按兵不动。——查理·芒格。 人生就像滚雪球，重要的是发现很湿的雪和很长的坡。——巴菲特。
人生算法：你面对世界不断重复的最基本的套路——罗振宇

最后还是以木心先生的一句话“岁月不饶人，我又何尝绕过岁月”结尾
春耕秋收，用心播种，好好栽培，辅以岁月，自当收获累累果实
