2018时间的朋友演讲小结

更新于: 2019-01-09 | 分类于 感悟 | | 阅读次数:
字数统计: | 阅读时长 ≈

时间：2018年12月31日晚8点半，地点：深圳春茧体育馆

——开篇
1，你有你的计划，世界另有计划——万维钢
2，成年的滋味总是很复杂，熟悉的也许只能用来怀念，依赖的也许必须要放手。
3，我们必须习惯，这个世界抽换掉一些我们喜欢的、熟悉的东西。
4，真实的世界里，并不存在抽象的两难选择。每时每刻，我们做事的人面对就是一张时间表，就是这张时间表上具体的时间安排而已。
5，做事的人无所谓悲观还是乐观，我们只关心如何让把事做好。
6，我们是伞兵，伞兵，天生就是被包围的。
7，对做事的人来说，什么时候不难？对不做事的人来说，难不难跟他有关系吗？
8，一个做事的人，总要以某种方式确认在这个世界的存在。
9，这个世界的模型，不是大河，而是多米诺骨牌。

——小趋势
1，凡我赶不上的，我就做好准备，到未来等它。
2，小趋势不是大趋势的小时候。小趋势是影响趋势的趋势，带来改变的改变。
3，等号的意思是，有些东西不重要。
4，坏消息是，每一班车停靠的时间都很短，而且看起来都像末班车。好消息是，真的没有末班车，车是一会儿一趟，越来越密。

——我能看到事实吗？
1，真相，在越来越远离我们。
2，一个人能同时保有全然相反的两种观念，还能正常行事，是第一流智慧的标志。
3，千万别把自己当主流，你要对自己的趣味保持一种警醒和谦卑。千万不要动不动就以己度人。
4，在中国，再众所周知的事情，都起码有一个亿的人不知道，而大多数时候，是十个亿的人都不知道。
5，不抽象，就无法深入思考。不还原，就看不到本来面目。
6，还原能力是最稀缺的能力。
7，制造业是我们构建良好社会的压舱石。
8，没有一个人能靠做空自己祖国而成功的。

——我能感知“非共识”吗？
1，创新的本质就是“非共识”——从脱离共识到再造共识——梁宁
2，决定我们这一代人个体命运的，除了众所周知的大趋势，更需要我们每一个人自己去主动发现的小趋势。
3，非共识从来不反对什么，它知识把被忽略的东西呈现出来而已。
4，看着隔层纱，其实隔座山。
5，我出生就有的科技，本该如此；我年轻时候的科技，伟大革命；我中年之后的科技，异端邪说。——道格拉斯·亚当斯的科技三定律。
6，人类的很多古老共识，并没有沉底，只是在潜水。
7，一个人也想好好生活。
8，所谓的创新，就是要回到一个古老的事物，但是用全新的手段把它再做一遍。所谓的非共识，就是你刚开始不认得，但是到最后，原来早已相识。
9，若不进入传统，则无法添加新物。
10，有一种创新叫往回走。
11，庆典、征服、竞技，三个永恒的让人欢呼的主题。
12，竞，就是它所回到的那个母体。电，就是我们这个时代对它的加强。

——我们的时间够用吗？
1，所谓信息过载，是你对环境的熟悉程度太低——万维钢。
2，这不是结束，甚至不是结束的开始，只是开始的结束。——丘吉尔。
3，在百岁人生的坐标下，咱得分得清什么能穿越时间，什么会被时间过滤。所有能穿越时间的东西，就该坚守，因为人生太长了。所有会被时间过滤的，该翻篇就翻篇。
4，你在朋友圈里又佛又丧，你在收藏夹里偷偷地积极向上。

——我的力量从哪里来？
1，人对人非常直接的感知和判断，胜过千万条数据。
2，信用飞轮。这个世界绝不会只变成机器的世界，算法的世界，代码的世界。这个世界在很多场景下，完全呈现为一个人推动人的世界。

——时间愿意和我做朋友吗？
1，普通人的努力，在长期主义的复利下，会积累成奇迹。
2，长期主义不仅要坚持你想做的事情，而且不能中断你在做的事情，还要持续地不被诱惑。一旦中断，前功尽弃。
3，长期主义是微弱信号的放大器。
4，用自己的超级确定性，来对冲外界的不确定。
5，成就=核心算法大量重复动作大量重复动作。

——尾声
1，一个做事的人，怎么会怕变化呢？在变化面前，我是巴不得的啊。如果这个世界没变化，还有什么机会轮得着我吗？
2，有意瞄准，无意击发。
3，历史一直往前走，但他有时候是一个反转大师。反转大师提供的机会，你不能辜负。——何帆。
4，永远不要低估中国人身上那种朴素的、但是又强烈的向上冲动。民族的精神红利。
5，所有事到最后都会是好事。如果还不是，那它还没到最后。——约翰·列侬
6，对未来最大的慷慨，是把一切献给现在。——阿尔贝·加缪

时光倏忽，一年一眨眼就过去了，过去的一年是充实和虚无同时存在的一年，有果断，也有犹豫，时而充满斗志，时而又很颓丧，不过好在一直坚持在读书，好在有得到一直在陪着我。新的一年里，要更加的积极一些，不管是在得到的学习上，还是在事业中，在感情里，用一次次小小的努力，去勇敢地坚持，去相信长期主义的奇迹

有趣的都打赏了~
