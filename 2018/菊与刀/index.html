《菊与刀》：矛盾的日本人

更新于: 2018-08-02 | 分类于 阅读 | | 阅读次数:
字数统计: 1,323 | 阅读时长 ≈ 4

菊花是日本皇室的象征，而刀是武士道的象征，一个柔美，一个刚烈，一对互相对立的事物，正好可以用来形容日本整个民族矛盾的性格和心理。《菊与刀》是一本研究日本民族性格的经典书籍，它最开始只是用来指导美国在二战后如何决定对日政策，后来慢慢成为研究日本的必读文献，它的影响之大，可见一斑。

1

日本社会有浓厚的报恩文化，一旦欠下别人的恩情，就会想法设法去报恩，即使是要付出生命的代价，也在所不辞。因为整个社会都有很重的报恩心理负担，所以整个日本民族都在日常生活中小心翼翼地避免让别人背负上一种恩情。他们总希望自己的天平能够在欠恩和报恩之间保持平衡，否则他们内心会一直处在纠结自责的状态。在日本，甚至有一条法律规定：“遇到争端，无关者不得干预”就是出于这种报恩文化的影响，避免因为帮忙而让人陷入报恩的负担之中，而一旦日本人决定报恩，就几乎是毫无原则的。 对日本人来讲，突然受到生疏者的恩惠是最讨厌的事。

日本人的这种报恩文化体现在报答皇恩上，更是明显。在中国，我们经常说“皇恩浩荡”，但是我们报答皇恩起码还得有一个前提，就是皇帝必须是个好皇帝，臣民的忠必须建立在皇帝的仁上面。但是在日本，却是可以忽略仁的，不管天皇仁还是不仁，日本人报答天皇的恩情都是毫无原则的。中国几千年来，朝代更迭无数个，但是日本却是万世一系，天皇从来就没有被推翻过，全是他们家在统治日本，就是这种不加反思的报恩思想导致的。

2

在大多数文化里，一个人做错事了，不论别人是否知道，他们的内心都会产生罪恶感，会主动去道歉或者尝试去补偿错过。 但是日本人却有另一种文化——耻感文化，他们不太讲究一件事本身的是非对错，而是看重这件事给自己带来的羞耻感。也就是说，如果做错事别人不知道，不去羞辱自己，就可以当这件事没有发生过，而不会自己主动去道歉或补偿。所以当一件不好的事情发生以后，他们想的可能是极力先掩饰和否定，避免让别人知道。比如很多日本人拒绝承认对亚洲人民的屠杀。

每个民族在面对侮辱的时候反应都不太相同，我们中国人可能就觉得，被侮辱不算什么大事，如果反应过激只能说明自己气量狭小，“宰相肚里能撑船”，大人物都有容人之量。泰国人面对侮辱会听之任之，因为这样可以暴露侮辱者的人格。但日本人面对侮辱，反应就很激烈。他们觉得，只要受到的侮辱没有报复回去，这个世界就不平稳，必须努力使世界恢复平衡。所以他们认为复仇是美德，而不是罪恶。

骨子里深深的耻感文化，使得看重名誉的日本人，有着世界上最高的自杀率。直到今天，仍有人在约定的时间里自杀，只是为了维护自己的名誉。

3

除了奇怪的报恩文化和耻感文化，日本民族还有奇怪的道德观念。

日本人对感官享乐特别宽容(可以想想拍那啥片儿为什么就日本是合法的)。因为在日本人看来，肉体根本不是罪恶，追求肉体的快感没有任何可以感到羞耻的地方。

日本人就没有强烈的善恶观，因为在他们看来，一些国家的人之所以特别强调道德，那是因为，他们的劣根性需要用道德观念加以约束。

日本人把履行义务规定为人生的最高任务。所以，一旦有需要，他们可以随时放弃所有肉体的享乐，去完成自己的使命，比如二战时，神风特攻队，不惜一切，只为履行义务。 而只要不和最高义务相冲突，任何肉体和感官的享乐都没有必要贬低和排斥。正是因为这些原因，日本才会有那么发达的色情产业，而且，僧侣也是可以结婚的。

《菊与刀》从文化和性格的角度分析了日本民族，既礼貌待人又粗鲁好斗，既喜欢菊的柔美又崇尚武士精神，既保守顽固又乐于尝试新事物，既传统礼佛又色情荒诞，这种矛盾的民族性格，养成了日本这个矛盾的民族。
